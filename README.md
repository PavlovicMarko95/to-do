# JS/Python ToDo App

## Description
Web application for managing tasks with three tabs: All, Active, Completed.  
Frontend built with JavaScript, backend with Python (Flask), tasks stored in a JSON file.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- View all, active, or completed tasks

## Technologies
- HTML, CSS, JavaScript
- Python, Flask
- JSON file for task storage

## How to Run
1. Install Flask: `pip install flask flask-cors`  
2. Run the server: `python app.py`  
3. Open `http://127.0.0.1:5000/` in your browser# to-do
